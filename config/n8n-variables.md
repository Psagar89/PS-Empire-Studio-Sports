# 🔧 n8n Variables Setup

n8n में **Settings → Variables** में जाकर ये सब variables add करो:

---

## Required Variables

| Variable | Description | Example |
|----------|-------------|---------|
| `NEWSAPI_KEY` | NewsAPI.org API Key | `a1b2c3d4e5f6...` |
| `GEMINI_KEY` | Google Gemini API Key | `AIzaSy...` |
| `FOOTBALL_DATA_KEY` | Football-Data.org API Token | `xyz123...` |
| `SHOTSTACK_KEY` | Shotstack API Key | `abc-def-ghi...` |
| `IG_USER_ID` | Instagram Business Account ID | `123456789` |
| `IG_ACCESS_TOKEN` | Facebook Page Access Token | `EAA...` |
| `GOOGLE_DRIVE_FOLDER_ID` | Google Drive Folder ID | `1ABCxyz...` |

---

## How to Add Variables in n8n

1. n8n dashboard खोलो
2. Left sidebar में **Settings** (⚙️) पर क्लिक करो
3. **Variables** टैब चुनो
4. **Add Variable** बटन दबाओ
5. Key और Value भरो
6. **Save** करो
