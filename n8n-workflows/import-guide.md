# 📥 How to Import n8n Workflows

## Method 1: Import from File (Easy)
1. Open your **n8n dashboard** (Render URL)
2. Click **+ Workflow** or open existing
3. Go to **Workflow Settings** (top right ⚙️ icon)
4. Click **Import from File**
5. Select the JSON file from this folder
6. Fill in your API keys (see `../config/n8n-variables.md`)
7. Toggle **Active** switch ON
