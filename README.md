**PS Empire Studio Sports** is a fully automated, AI-driven sports content channel that posts daily:
- 🤣 **Funny Football Moments**

Across **YouTube Shorts, Instagram Reels, and TikTok** — all created by AI, zero manual work.

## 🚀 What This Repo Contains

| Folder | Description |

| `n8n-workflows/` | Import-ready n8n automation workflows |

## ⚙️ Tech Stack (100% Free)

| Tool | Purpose |
|------|---------|
| **n8n** (hosted on Render) | Workflow automation engine |
| **Google Gemini API** | AI script writing |
| **Shotstack API** | AI video rendering (text-to-video) |
| **NewsAPI** | Fetch funny football news |
| **Football-Data.org** | Fetch match results & data |
| **YouTube Data API v3** | Auto-upload Shorts |
| **Instagram Graph API** | Auto-post Reels |
| **GitHub** | Store, version, and share everything |
